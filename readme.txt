== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install King Composer plugin then install  CC Addons For King Composer.

Installation details are here also <a href="http://www.janealam.com/wordpress-plugin-documentation-cc-addons/">Installation</a>

This addon plugin works like other page builder addon plugin. Go to new page then click on King Composer builder then click on Add Element you should see King Composer Extensions All In One menu  as Name "Creative Code Addons" on Manu Top, click on it you will see All King Composer Extensions will Open . Open it and browse an image and customize according to your needs.
